# 010: Remove LOCAL_API_KEYS_JSON and Fix Startup

## Контекст
LOCAL_API_KEYS_JSON был временным решением для MVP, но теперь у нас есть админка для управления ключами. Этот механизм создаёт проблемы:
- IntegrityError при дублирующихся ключах
- Запутанная логика startup
- Не нужен если ключи создаются через админку

4. [x] ADMIN_PASSWORD генерится если не задан (ранее сделано)
