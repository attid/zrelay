# 007: Connect Admin Dashboard to FastAPI

## Контекст
Админка реализована в `src/interface/admin/app.py`, но не подключена к основному FastAPI приложению. В продакшене админка недоступна.

## План изменений
1. [ ] Подключить `admin_app` в `src/interface/api/main.py`:
   - [ ] Добавить `app.include_router(admin_app, prefix="/admin")` или `app.mount("/admin", admin_app)`
2. [ ] Проверить работу админки в dev-режиме:
   - [ ] `/admin` доступен
   - [ ] Авторизация работает
   - [ ] HTMX-взаимодействие работает (создание/удаление ключей)
3. [ ] Проверить в docker-compose:
   - [ ] Админка доступна внутри контейнера

## Риски и открытые вопросы
- Starlette/FastAPI совместимость с FastHTML (проверялось ранее, должно работать).

## Верификация
- `curl http://localhost:8000/admin` возвращает HTML админки
- Логирование ключей через админку записывается в БД
