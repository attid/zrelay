# Golden Principles: zrelay

1. **Security First**: Upstream API ключи никогда не возвращаются во внешних ответах и не логгируются.
2. **Standard Error Format**: Все ошибки возвращают JSON по единому формату (`{"error": {"code": "...", "message": "..."}}`).
3. **Traceability**: Каждый запрос имеет уникальный `request_id`, передаваемый сквозь все слои.
4. **Stateless Logic**: Шлюз хранит состояние только в БД (ключи и логи), сама обработка запроса независима от состояния сервера.
5. **No Hallucinations**: Если инструмент вернул ошибку, шлюз должен передать её корректно, а не пытаться скрыть или интерпретировать.
6. **Atomic Plan Completion**: Перенос плана из `active/` в `completed/` должен быть закоммичен до завершения задачи — последний коммит обязан включать перемещённый файл в `completed/`.
